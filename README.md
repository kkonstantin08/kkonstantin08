## Featured projects

### [PlanSync](https://github.com/kkonstantin08/se-toolkit-hackathon)

A full-stack weekly planner for students with tasks, events, reminders, recurring routines, AI-assisted input, and Google Calendar integration.

### [3D Solar System](https://github.com/kkonstantin08/3d-solar-system)

An interactive browser-based 3D visualization of the Solar System built with Three.js, with camera focus tracking and a responsive interface.

### [Differential-drive robot](https://github.com/kkonstantin08/Differential-drive-robot)

A robot vacuum simulation with differential drive, autonomous movement using a snake algorithm, and real-time trajectory visualization.

### [Clicker](https://github.com/kkonstantin08/clicker)

A mobile-oriented Vue 3 + Vite clicker app designed in the style of a Telegram mini app.

### [Horoscope Telegram Bot](https://github.com/kkonstantin08/horoscope_tg_bot)

A Telegram bot that fetches daily horoscopes, translates them into Russian, and sends them through a simple button-based interface.

### [Parsers](https://github.com/kkonstantin08/parsers)

A collection of modular web scrapers for automated data collection with flexible output formats such as CSV, JSON, and databases.
