## Selected work

- **[PlanSync](https://github.com/kkonstantin08/se-toolkit-hackathon)** — a full-stack weekly planner for students with reminders, recurring routines, AI-assisted input, and Google Calendar integration.
- **[3D Solar System](https://github.com/kkonstantin08/3d-solar-system)** — an interactive browser visualization of the Solar System built with Three.js.
- **[Differential-drive robot](https://github.com/kkonstantin08/Differential-drive-robot)** — a robot vacuum simulation with autonomous navigation and real-time trajectory rendering.
